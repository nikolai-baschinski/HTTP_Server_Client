This HTTP Server and Client uses the cpp-httplib (https://github.com/yhirose/cpp-httplib)

1. Clone the repo
2. Open the Visual Studio solution file HTTP_Server_Client.sln twice
3. In the first one configure Server as startup project
4. In the second one configure Clinet as startup project
5. Run the Server then the Clinet applications
